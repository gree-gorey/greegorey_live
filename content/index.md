## что это такое и зачем оно нужно
### зачем?
Это один из способов структурировать свои мысли/заметки/посты и т.п. Я это использую и для своих личных заметок, а это вариант для публичных постов, из которых тоже можно построить интересный граф знаний.
### почему такой формат
Есть много разных интрументов для этого (Notion, Obsidian), я решил попробовать [Obsidian](https://obsidian.md/):
* простота: все заметки это просто файлы md, без привязки к приложению;
* поиск по тегам и ссылки как в вики на все файлы;
* визуализация в виде графа;
* можно подключать плагины, в том числе для вызова LLM и делать запросы к своим заметкам;
* можно бесплатно хостить на GitHub
